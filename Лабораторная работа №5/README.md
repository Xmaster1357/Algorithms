# Бинарное дерево поиска (BST)

## Реализованные функции:

`insert(root, value)`

Добавляет новое значение в дерево поиска.

`search(root, target)`

Выполняет поиск элемента в дереве.

Возвращает:

- True, если элемент найден;
- False, если элемент отсутствует.

`inorder(root)`

Выполняет симметричный обход дерева.

`count_leaves(root)`

Подсчитывает количество листьев дерева.

### Пример дерева

После вставки элементов:

`50, 30, 70, 20, 40, 60, 80`

будет построено дерево:

        50
       / \
     30   70
    / \   / \
    20 40 60 80

_Симметричный обход выведет:_

`20 30 40 50 60 70 80`

_Количество листьев:_

`4`

_Листья:_

`20, 40, 60, 80`
