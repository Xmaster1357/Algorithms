# Bus Route — Linked List

## Описание

Программа реализует автобусный маршрут с помощью односвязного списка.

Поддерживаются:
- добавление остановок;
- подсчет времени маршрута;
- поиск остановки через N шагов;
- разворот маршрута;
- кольцевой маршрут;
- сохранение отчета в файл.

---

# Почему LinkedList

Односвязный список хорошо подходит для маршрута, потому что:
- остановки идут последовательно;
- удобно добавлять элементы;
- легко реализовать кольцевой маршрут;
- память выделяется динамически.

Недостаток:
- медленный доступ по индексу (`O(n)`).

# Основные классы

## BusStop
Хранит:
- название;
- координаты;
- время до следующей остановки;
- ссылку на следующую остановку.

## BusRoute
Управляет маршрутом автобуса.

---

# Документация функций

## add_stop(...)
Добавление остановки.

## total_route_time()
Подсчет общего времени маршрута.

## get_stop_after_n(n)
Поиск остановки через N шагов.

## reverse_route()
Разворот маршрута.

## make_circular()
Создание кольцевого маршрута.

## generate_report(filename)
Создание текстового отчета.

## print_route()
Вывод маршрута в консоль.

---

# Обработка ошибок

Обработаны:
- пустой маршрут;
- выход за границы;
- кольцевой маршрут;
- одна остановка.

---

# Пример

```python
route = BusRoute()

route.add_stop("Центр", (55.75, 37.61), 10)
route.add_stop("Парк", (55.76, 37.62), 7)

route.print_route()
```
