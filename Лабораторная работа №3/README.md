# Описание программы

Программа реализует очередь задач с возможностью:

- добавления задачи
- удаления задачи
- просмотра первой задачи
- проверки очереди на пустоту
- вывода всех задач

---

# Класс Task

Класс `Task` используется для хранения информации о задаче.

## Поля класса

| Поле        | Описание             |
| ----------- | -------------------- |
| id          | идентификатор задачи |
| description | описание задачи      |
| priority    | приоритет задачи     |

---

# Класс TaskQueue

Класс `TaskQueue` реализует очередь задач.

## Методы класса

| Метод         | Описание                    |
| ------------- | --------------------------- |
| enqueue(task) | добавление задачи в очередь |
| dequeue()     | удаление первой задачи      |
| front()       | просмотр первой задачи      |
| isEmpty()     | проверка очереди на пустоту |
| show_tasks()  | вывод всех задач            |

# Вариативная часть №9

Добавлена проверка на одинаковые id задач.

Если задача с таким id уже существует, программа выводит сообщение об ошибке и не добавляет задачу в очередь.

Пример:

```text
Ошибка: задача с id = 1 уже существует!
```
