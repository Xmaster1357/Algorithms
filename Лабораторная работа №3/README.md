# Лабораторная работа: Система обработки задач (Queue)

## Описание

В рамках лабораторной работы реализована очередь задач на языке Python.

Каждая задача содержит:

- идентификатор (`id`)
- описание (`description`)
- приоритет (`priority`)

Для хранения задач используется структура данных **Queue (очередь)**.

---

## Возможности программы

- добавление задач в очередь;
- удаление задач из очереди по принципу FIFO (First In, First Out);
- просмотр первой задачи без её удаления;
- проверка очереди на пустоту;
- получение количества задач в очереди;
- вывод всех задач;
- запрет добавления задач с одинаковым идентификатором.

## Класс Task

Класс предназначен для хранения информации о задаче.

### Поля

| Поле | Тип | Описание |
|--------|--------|--------|
| id | int | Уникальный идентификатор задачи |
| description | str | Описание задачи |
| priority | int | Приоритет задачи |

---

## Класс Queue

Класс реализует очередь задач.

### Поля

| Поле | Тип | Описание |
|--------|--------|--------|
| items | list | Список задач, находящихся в очереди |

### Методы

| Метод | Описание |
|--------|--------|
| enqueue(task) | Добавляет задачу в очередь. Перед добавлением проверяет уникальность идентификатора |
| dequeue() | Удаляет и возвращает первую задачу из очереди |
| front() | Возвращает первую задачу без удаления |
| is_empty() | Проверяет, пуста ли очередь |
| size() | Возвращает количество задач в очереди |
| show() | Выводит все задачи, находящиеся в очереди |
