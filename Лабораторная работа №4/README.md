# Сокращатель ссылок

## Структура проекта:
```text
.
├── shortener.py
├── test_shortener.py
└── README.md
```

## Возможности программы:

- добавление длинной ссылки;
- автоматическая генерация короткого кода;
- получение длинной ссылки по коду;
- проверка существования кода;
- просмотр всех сохранённых ссылок;
- подсчёт количества переходов по каждой ссылке;
- автоматические тесты.

#### Запуск программы

```bash
python shortener.py
```

#### Запуск тестов

```bash
python test_shortener.py
```
